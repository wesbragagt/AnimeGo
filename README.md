## Description
  AnimeGo! is a site which allows you to browse through anime recommendations and add animes to your watch list.
  Under development include a login and sign up system.

## Technologies Used

  ### Front-end  
  - Bootstrap MD
  - AJAX
  - JQuery
  
  ### Back-end
  - Nodejs
  - Express
  - Sequelize
  
  ### Database
  - MysQl
  
  ### Testing
  - Chai
  - Mocha

  ### Deployment
  - Heroku
